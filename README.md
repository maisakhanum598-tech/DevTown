# DevTown Mini Project – Amazon Sales Analysis

This repository contains my DevTown Mini Project, where I analyzed Amazon sales data using Python and Excel.  
The project focuses on identifying key business insights, understanding customer behavior, and presenting clear visual dashboards.

---

## 📌 Project Overview

The goal of this project is to explore Amazon’s sales dataset and uncover meaningful patterns such as:

- Top-performing products and categories  
- Monthly sales and profit trends  
- Customer preferences and payment behavior  
- High and low-performing states  
- Actionable recommendations for improving sales and profitability  

---

## 🛠️ Tools & Technologies

- Python (Pandas, NumPy, Matplotlib)  
-  Google Colab  
- Microsoft Excel (Pivot Tables, Pivot Charts, Slicers)  
- GitHub for version control  

---

## 📈 Excel Dashboard

The Excel dashboard includes:

- Pivot charts for **State**, **Category**, and **Monthly Sales**  
- Slicers for interactive filtering  
- KPI boxes showing **Total Sales**, **Total Orders**, and **Average Sales**  

This dashboard provides a clear and interactive overview of the dataset.
